## Project info

# Task Manager

A simple task-tracking web application built with Vite, React and TypeScript.

## Project overview

This repository contains the source for a local task manager application — create, edit, complete, and organize tasks.

## Requirements

- Node.js (20+ recommended)
- npm (or yarn/pnpm)
- Git

On Windows you can install Node.js from https://nodejs.org/ or use nvm-windows.

## Quick start (Windows)

Open PowerShell or a terminal:

```powershell
# Clone the repository
git clone <YOUR_GIT_URL>

# Enter the project directory
cd task-manager

# Install dependencies
npm install

# Start the development server
npm run dev
```

Visit http://localhost:8080 (or the URL shown in the terminal) to view the app.

## Available scripts

- npm run dev — start development server
- npm run build — create production build
- npm run preview — locally preview production build
- npm test — run unit tests (if configured)
- npm run lint — run linter (if configured)

Adjust commands to yarn or pnpm if you prefer.

## Project structure (example)

- src/ — application source
  - components/
  - contexts/
  - pages/
  - hooks/
  - utils/
  - styles/
- public/ — static assets
- vitest.config.ts or jest.config.js — test config
- tsconfig.json — TypeScript config
- package.json — scripts and dependencies
